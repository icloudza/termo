# 安全策略 / Security Policy

## 支持的版本 / Supported versions

始终修复**最新发布版本**中的安全问题。旧版本请先升级到最新版。
Security fixes target the **latest released version** only. Please update before reporting.

| 版本 / Version | 支持 / Supported |
|---|---|
| 最新版 / Latest | ✅ |
| 更早 / Older | ❌ |

## 报告漏洞 / Reporting a vulnerability

**请勿公开提交安全漏洞的 Issue。**
**Please do not open a public issue for security vulnerabilities.**

请通过 GitHub 私密渠道报告 / Report privately via GitHub:

- 打开 [私密安全公告 / private security advisory](https://github.com/icloudza/termo/security/advisories/new)

报告请尽量包含：受影响版本、复现步骤、影响评估、以及可能的修复建议。
Please include: affected version, reproduction steps, impact assessment, and any suggested fix.

我们会尽快确认收到并跟进；修复发布前请勿公开细节。
We will acknowledge and follow up promptly; please keep details private until a fix ships.
