# 更新日志

本项目所有值得记录的变更都会写在这里，遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/)。
发版由 `scripts/release.sh` 自动把下方 `[Unreleased]` 归档为对应版本，并同步到 GitHub Release、软件更新弹窗与 Sparkle appcast。

## [Unreleased]

## [0.9.2] - 2026-07-01
- 移除启动行为中未实现的「恢复上次会话」选项
- RDP 打开新窗口时默认不再自动全屏
- 终端默认字号调整为 13
- 自动检查更新默认开启，检查频率提升为每小时一次
- 主机 / RDP 分组下拉支持再次点击取消选择
- RDP 主机列表现按分组分区显示

## [0.9.1.1] - 2026-07-01
- 移除内置的演示主机与模拟监控数据
- 重做「软件更新」弹窗：改用原生标题栏，并在关于页内联检查更新
- 修复检查更新后无法再次点击的问题
- 补充出口加密合规声明（仅使用标准 SSH/TLS 加密）

## [0.9.1] - 2026-06-30
- 首个公开发布版本（Developer ID 渠道，支持应用内自动更新）
- 远程桌面（RDP）：全彩图形管线（Progressive/RemoteFX）、键盘输入、剪贴板双向同步、动态分辨率
- SSH / SFTP / 终端 / 端口转发 / 密钥：全栈进程内 libssh2，自包含、不依赖系统 ssh
- 主机监控、端口转发、代码片段、SSH 密钥管理
- 应用内自动更新（Sparkle）
